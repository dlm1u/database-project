# Автоматизированная система логистики склада

---

## О проекте
 
Данная база данных предназначена для автоматизации учета товаров на складе, отслеживания их перемещений между зонами и ячейками, а также контроля работы персонала с входящими поставками и исходящими заказами.

*По мере заполнения проекта, информация будет дополняться*
---

## Архитектура проекта
Проектирование выполнено в три этапа:

1.  **Концептуальная модель:** Определение основных сущностей и связей.
2.  **Логическая модель:** Нормализация данных, определение первичных и внешних ключей.
3.  **Физическая модель:** Определение типов данных и ограничений.

> Все схемы доступны в директории `./schemes`

## Структура базы данных
В базе реализовано 6 таблиц:
* `CLIENT` — информация о клиенте.
* `PRODUCT` — перечень товаров.
* `WAREHOUSE_ZONE` — сведения о складах.
* `SUPPLIER` — данные о поставщиках.
* `EMPLOYEE` — Информация о сотрудниках.
* `ORDER` — сведения о заказах.
